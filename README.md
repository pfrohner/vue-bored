# Bored App

## Demo
https://pfrohner.github.io/vue-bored/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production.
```
npm run build
```

### Builds and deploys to github.io
```
npm run deploy
```

### Run your tests
```
npm run test
```
